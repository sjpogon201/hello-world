Neighborhood Map

The Neighborhood Map Project is app that displays the recommended locations to visit around an area. My app is centered around the locales in Downtown Chinatown in Mahattan, NY. This project utilizes the Google Maps API, and third party API of OpenWeather API to provide weather forecast as well as the Foursquare API to provide location details.

Item Catalog

This app implements CRUD functionality to create, showcase, edit, and delete movies within the database.

Multi-User Blog

This is a project created through Udacity's Full Stack Nanodegree to have an intro to backend web developement programming. The project consists of a multi-user blog where users can sign in to create posts or comment on other posts as well as like other user's posts. This Multi-user Blog will combine the coding skills of python to execute the front-end actions and GQL (Google's Query Language) to perform saving/exchange of data in the backend. Movie Trailer App

This is a generated static web page that vistors can go to browse my favorite movies. Each movie will its movie poster, movie synopsis, and movie trailer once you click on the movie poster. This program was created using Python and the useful Python library "Fresh Tomatoes". Fresh Tomatoes creates a static movie page based on the data you input into file when calling the functions that are inside this library.

Banking Android App

This is an Android app that was part of a project for the class CET 3640. The Android app is a banking app that allows the user to show balance of banking accounts as well as withdraw, deposit, pay bills/loans with the option of paying via checking or savings account.

Quizzical

Quizzical is an hybrid mobile app that tests the users knowledge on pop culture such as Music, Movies, and Technology. This app was created using the aforementioned three web technologies (HTML5, CSS, JavaScript), Ionic Framework, Node.Js, and Apache Cordova. Through the flexibility Apache Cordova allows in the app creation process, the app is available on all mobile OS platforms such as Android, iOS, Windows, and etc. This was app was part of research project with the paper being titled "Developing Hybrid Mobile Applications With Ionic".

Bi-Ped Walking Arduino Robot

BoB the Walking Biped is a bi-ped robot that has the ability to work on two legs in any direction built using Arduino. The robot has various integrated features such as Object Detection/Avoidance Systems, Bluetooth Remote Control, and IR Detection.

Link to BoB the Biped

BoB the Walking BiPed was inspired by the BoB the Walker design as found on the Make website. This robot build was part of a project for the class CET 4811.
